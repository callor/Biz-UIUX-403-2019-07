이벤트 핸들링

만약 버튼을 클릭하는 동작을 브라우저가 감시하고 있는 상황이되면 버튼의 클릭 동작을 수시로 살펴야하는 불편한 상황이 발생을 할 것이다 이런 상황을 우리는 blocking 상태라고 하며 윈도우 이전의 운영체제, console 환경의 어플리케이션에서는 가능한 일이나 윈도우처럼 멀티테스팅, 멀티스레딩 환경에서는 절대 해서는 안되는 일이다

그래서 운영체제 차원에서 이벤트(event)라는 것을 지원하며 아래처럼 button에 onclick이라는 속성을 지정해주면 운영체제가 해당 버튼의 클릭 여부를 감시하고 있다가 클릭 동작이 발생하면 설정된 함수를 실행해준다

이벤트가 발생했을때 동작하는 함수, 객체, 어떤 것들을 이벤트 핸들러 라고한다.
